HPE Helion Stackato Backup and Restore Plugin
 
The Backup and Restore Plugin for Cloud Foundry CLI allows Operators to backup Helion CF related meta-data (Users, Org, Space etc) as well as Application bits. Operators can leverage the CLI to restore HCF state based on backup snapshot.
 
Further documentation can be found at: http://docs.hpcloud.com/permalink/stackato/4.0/home
